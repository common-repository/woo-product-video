== Installation ==

The quickest method for installing the product video plugin

1. Visit Plugins -> Add New -> Choose your plugin zip from your directory
2. Click "Install Now"
3. Finally click "Activate"

If you would prefer to do things manually then follow these instructions:

1. Upload the `woo-product-video` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click "Activate"
